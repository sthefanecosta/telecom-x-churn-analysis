# Análise de Evasão de Clientes (Telecom X)

Este projeto apresenta uma análise exploratória sobre o Churn (evasão) de clientes da Telecom X. O objetivo é identificar os principais perfis de clientes que cancelam o serviço e oferecer recomendações para aumentar a retenção.

## 📝 Sobre o Projeto
Durante a análise, foquei em entender quem está saindo e por quê. O trabalho envolveu desde a limpeza inicial dos dados, transformando informações brutas em métricas úteis, até a geração de gráficos que mostram o comportamento desses clientes.

## 🛠️ Estrutura
* `Analise_Evasao_TelecomX.ipynb`: Notebook completo com o tratamento dos dados, cálculos estatísticos e visualizações gráficas.
* `README.md`: Este arquivo com o resumo e instruções.

## 📈 Principais Insights
* **Contratos:** Clientes com contratos mensais (Month-to-month) têm a maior taxa de cancelamento.
* **Tempo de Casa:** Quem cancela o serviço geralmente tem um tempo de permanência menor (média de 18 meses) comparado a quem permanece.
* **Financeiro:** Clientes que saem apresentam um gasto mensal médio mais alto.

## 🚀 Como executar
1. Abra o arquivo `Analise_Evasao_TelecomX.ipynb` no [Google Colab](https://colab.research.google.com/).
2. Certifique-se de carregar sua base de dados antes de rodar as células.
3. Execute o código sequencialmente para reproduzir a análise e os gráficos.
